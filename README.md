## ✨ 这是什么？

这是一个基于**女娲造人蒸馏法（Huashu Nuwa）**构建的 AI 思维视角 Skill。它不是简单的"人设扮演"，而是从 6 个维度系统采集、提炼、验证后形成的**可加载认知框架**：

- **5 个心智模型** — 实践论、矛盾分析法、人民主体论、以弱胜强论、自觉辩证法
- **10 条决策启发式** — 从"没调查不做决策"到"敢于冒险博取战略利益"
- **5 套表达风格指纹** — 比喻降维、短句断言、三段排比、爱憎分明、古典底色
- **6 条核心价值观 + 诚实边界** — 包含 5 大内在矛盾与使用注意事项
- **5 张速查卡片** — 面对强对手/复杂局面/说服/犯错/写作的快速反应模板

## 🧩 核心能力一览

### 心智模型

```
实践第一论 → 正确思想只能从社会实践来
矛盾分析法 → 抓住主要矛盾，其他问题迎刃而解
人民主体论 → 兵民是胜利之本
以弱胜强论 → 纸老虎：战略藐视，战术重视
自觉辩证法 → 好事坏事在一定条件下互相转化
```

### 决策启发式（精选 Top 5）

| # | 启发式 | 经典案例 |
|---|--------|---------|
| 1 | 没调查过的决策不做 | 反例：大跃进 |

## 🚀 如何使用

### 方式一：WorkBuddy Skill（推荐）

在 WorkBuddy 中直接调用：
```
使用 skill: mao-zedong-perspective
然后输入你的问题，例如："用毛泽东视角分析一下当前AI行业的竞争格局"
```

### 方式二：直接阅读 SKILL.md

`SKILL.md` 是自包含的完整文档，可以直接阅读或集成到其他 AI Agent 框架。

### 方式三：参考调研报告

`references/research/` 目录下有 6 份详细的调研报告，总计约 2300 行，可作为深入研究的原始材料。

## ⚠️ 使用注意事项

**应该学习的**：
- 实践第一的认识论
- 矛盾分析法的思维工具
- 以弱胜强的竞争策略
- 深入浅出的表达能力
- 战略定力和耐心

**需要警惕的**：
- 当自信变成自负时
- 当对不同意见越来越不容忍时
- 当手段被目的正当性完全合理化时
- 当周围只剩下赞同声音时（信息茧房）

## 📊 质量评级说明

**★★★★☆ (4/5)**
