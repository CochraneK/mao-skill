## ✨ 这是什么？

这是一个基于**女娲造人蒸馏法（Huashu Nuwa）**构建的 AI 思维视角 Skill。它不是简单的"人设扮演"，而是从 6 个维度系统采集、提炼、验证后形成的**可加载认知框架**：

- **5 个心智模型** — 实践论、矛盾分析法、人民主体论、以弱胜强论、自觉辩证法
- **10 条决策启发式** — 从"没调查不做决策"到"敢于冒险博取战略利益"，每条带边界条件
- **5 套表达风格指纹** — 比喻降维、短句断言、三段排比、爱憎分明、古典底色
- **16 条口语化转化案例** — 从"讨老婆不要钱"到"天要下雨娘要嫁人"
- **8 项 AI 复现避坑指南** — 过度攻击/滥用排比/丢失辩证性等常见错误模式
- **6 条核心价值观 + 诚实边界** — 包含 5 大内在矛盾与使用注意事项
- **5 张速查卡片** — 面对强对手/复杂局面/说服/犯错/写作的快速反应模板
- **3 个完整分析范例** — 商业竞争 / 个人决策 / 团队管理（端到端输出样本）
- **💬 对话模式**（v1.3新增）— 第一人称角色扮演，4个对话场景 + 9条使用要点

### AI 复现避坑指南（8 大常见错误）

| # | 错误模式 | 正确做法 |
|---|---------|---------|
| 1 | 把"断言式"理解成骂人，满篇粗俗 | 断言 ≠ 粗鄙，批评犀利但有分寸 |
| 2 | 每段堆三句排比像喊口号 | **克制排比！**只在结尾或最高潮用一次 |
| 3 | 只输出强硬一面，不讲反面思考 | 核心是**辩证法**，每给判断都要包含反转可能 |
| 4 | 刻意用"之乎者也"装古典范儿 | 白话外壳，古意藏在用典和节奏里 |
| 5 | 全程严肃脸，像念悼词 | 再沉重也常有冷幽默穿插 |
| 6 | 用户要分析时只说正面不提教训 | 诚实边界是灵魂，不谈矛盾等于没用此Skill |
| 7 | 每段都引原话像考证文章 | **化用**而非堆砌——消化成自己的话说 |
| 8 | 一句话七八十个字从句套从句 | **短句短句短句！**五年级学生能看懂就对了 |

## 🚀 如何使用

### 📊💬 双模式切换

本 Skill 支持**两种模式**，按需自动切换：

| 模式 | 触发方式 | 效果 |
|------|---------|------|
| **📊 分析模式（默认）** | "用毛选分析一下..." / "帮我决策" / 直接提问任务 | 输出结构化分析报告 |
| **💬 对话模式** | "我想跟主席聊聊" / "主席您好" / "切换对话模式" | 第一人称角色扮演对话 |

> 💡 两种模式共享同一套心智模型和启发式规则，区别在于人称、结构和语气。

### 方式一：WorkBuddy Skill（推荐）

在 WorkBuddy 中直接调用：
```
使用 skill: mao-zedong-perspective

# 分析模式（默认）
→ "用毛泽东视角分析一下当前AI行业的竞争格局"

# 对话模式
→ "主席您好！我想跟您聊聊"
```

### 方式二：直接阅读 SKILL.md

`SKILL.md` 是自包含的完整文档（920行），支持双模式（分析+对话），可直接阅读或集成到其他 AI Agent 框架。

### 方式三：参考调研报告

`references/research/` 目录下有 6 份详细的调研报告，总计约 2300 行，可作为深入研究的原始材料：

| 01-writings.md | 核心著作提炼（《毛选》、哲学著作、诗词） | ~554行 |
| 02-conversations.md | 关键对话与言论（斯诺/斯特朗/蒙哥马利等） | ~511行 |
| 03-expression-dna.md | 表达风格深度拆解（修辞/节奏/幽默/口语化） | ~477行 |
