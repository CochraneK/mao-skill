## 一、高频用词和专属表达

### 1.1 核心政治词汇

| 词汇 | 出现语境 | 频次 |
|------|---------|------|
| **人民** | 几乎所有政治论述 | 极高 |
| **斗争** | 阶级斗争、路线斗争 | 极高 |
| **实践** | 认识论核心 | 高 |
| **矛盾** | 分析方法核心 | 高 |
| **实事求是** | 思想路线 | 高 |
| **群众路线** | 政治路线 | 高 |
| **独立自主** | 外交路线 | 中高 |
| **纸老虎** | 战略藐视 | 中 |

### 1.2 军事术语

| 词汇 | 含义 | 来源 |

| **游击战** | 战术原则 | 井冈山时期 |

### 1.3 日常口语化表达

- **"捣乱，失败，再捣乱，再失败，直至灭亡"** —— 帝国主义反动派的下场
- **"东风压倒西风"** —— 国际形势判断
- **"一切反动派都是纸老虎"** —— 战略上藐视
- **"贪污、浪费、官僚主义是极大的犯罪"** —— 三反运动
- **"下定决心，不怕牺牲，排除万难，去争取胜利"** —— 愚公移山精神

## 三、幽默与讽刺方式

### 3.1 自嘲式

- **"我是草头天子"** —— 调侃自己的湖南口音和农民出身
- **"我是一个打字员"** —— 与蒙哥马利谈话时自谦
- **"菩萨是农民立起来的"** —— 谈宗教本质
- **"我死了可以去见马克思"** —— 谈生死

### 3.2 讽刺式

- **"搬起石头砸自己的脚"** —— 批评对手的愚蠢
- **"瞎子摸鱼"** —— 批评脱离实际
- **"懒汉思想"** —— 批评等待思想
- **"夹紧尾巴做人"** —— 警告骄傲情绪

### 3.3 比喻式

| 比喻 | 含义 |
|------|------|
| 小石头砸大水缸 | 新生力量战胜腐朽力量 |
| 根据地=屁股 | 人没有屁股就不能坐下来 |
| 打扫房子/洗脸 | 批评与自我批评 |
| 种子与土地 | 党群关系 |
| 牛皮糖 | 积小胜为大胜 |

## 四、公开辩论风格

### 4.1 论战特点

| 特点 | 示例 |
|------|------|
| **以退为进** | "我们弱，但我们能赢" |
| **转化矛盾** | 劣势转化为优势 |
| **群众立场** | 站在"人民"一边 |
| **历史类比** | 用历史事件论证 |
| **排比增强** | "团结-批评-团结" |

### 4.2 经典论战

1. **与王明的路线斗争** (1935-1945)
   - 毛泽东：山沟里有马克思主义
   - 王明：教条主义照搬苏联

2. **与国民党的舆论战** (1945-1949)
   - "一切反动派都是纸老虎"
   - "为人民服务"

3. **与苏联的论战** (1960年代)
   - "修正主义"
   - "社会帝国主义"

## 五、语言风格演变

| 时期 | 风格特点 | 示例 |
|------|---------|------|
| **1920-1935** | 激进、煽动性 | "革命不是请客吃饭" |
| **1935-1949** | 成熟、辩证 | 《论持久战》《矛盾论》 |
| **1949-1957** | 建设性、乐观 | "六亿神州尽舜尧" |
| **1958-1966** | 理想主义、冒进 | "大跃进"相关表述 |
| **1966-1976** | 斗争哲学、极左 | 文革话语体系 |

## 六、诗词创作风格

### 6.1 意象体系

| 意象 | 象征意义 | 代表作品 |
|------|---------|---------|
| **红旗** | 革命、胜利 | 《长征》 |
| **青山** | 不屈意志 | 《十六字令》 |
| **风雪** | 艰难险阻 | 《沁园春·雪》 |
| **鲲鹏** | 宏大志向 | 《七律·和柳亚子》 |
| **蚂蚁** | 渺小敌人 | 《满江红》 |

### 6.2 艺术手法

- **浪漫主义**：夸张、想象（如"可上九天揽月"）
- **现实主义**：写实描写（如"百万雄师过大江"）
- **批判精神**：讽刺腐败（如《讥郭沫若》）

## 七、发现的矛盾

| 矛盾 | 说明 |
|------|------|
| **理论与实践** | 早期强调实事求是，晚期大跃进背离 |
| **言论与行动** | 说"批评与自我批评"，实际压制不同意见 |
| **反个人崇拜vs实际** | 早期反对神化，晚期默许个人崇拜 |
